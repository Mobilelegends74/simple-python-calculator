# Simple Python Calculator

This is a simple Python calculator that performs basic arithmetic operations including addition, subtraction, multiplication, and division. It is designed for educational purposes to demonstrate the usage of functions and classes in Python.

## Installation

To get started, clone this repository:

```bash
git clone https://github.com/yourusername/simple-python-calculator.git
Then install the necessary dependencies:

pip install -r requirements.txt

Usage

To use the calculator, you can run the main.py file. Example:

python calculator/main.py

Tests

To run the tests, simply execute the following command:

pytest tests/
